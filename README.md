# AlphaCred – Social-Fi Trade Call Reputation

AlphaCred is a full Social-Fi application built as a Canopy plugin. It allows users to publish trade calls, endorse others, resolve calls with real-time prices, earn reputation, and claim token rewards.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set environment variables: `TREASURY_WALLET`, `TWELVE_DATA_API_KEY`, `SECRET_KEY`
3. Run the plugin: `python plugin.py`
